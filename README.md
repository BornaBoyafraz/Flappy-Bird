# Flappy Bird

A modern space-themed Flappy Bird clone built with Python and Pygame.  
Features animated backgrounds, sci-fi obstacles, sound effects, scoring, and a clean UI.

## Features
- Parallax scrolling space background
- Animated spaceship player with tilt physics
- Sci-fi obstacle pillars
- Score + High Score tracking
- Start screen and Game Over screen
- Sound effects
- 60 FPS smooth gameplay

## Installation

### Requirements
- Python 3.10+
- pygame

### Install pygame
```bash
pip install pygame
```

### Run the game
```bash
python main.py
```

## Controls
- SPACE -> Jump / Start / Restart
- Q or ESC -> Quit
